Timestamp Microservice API Project

Instructions:
    Pass a string as a parameter, it will check to see if unix timestamp or natural language date is contained.
    If it does, returns both
    If not, it returns null for both

Example:

https://timestamp-ms.heroku.com/145929884 https://timestamp-ms.heroku.com/December%2031,%202020

Example Output:

{ "unix": 1459298840, "natural": "March 29, 2016" }
